Jul 16, 2026

Website rebuild, Perkeo/Lithium news, Oxygen rebranding and Krypton shelving.

The wednesware.github.io website has been rebuilt with a responsive multi-page layout for a more professional look and more satisfying feel.

The Perkeo programming language and its ecosystem, Lithium have received massive progress from the team.

There is no estimated time until Lithium 26.1, but the parser and interpreter are basically finished, and libraries and built-ins are now in development.

Oxygen (the terminal color and animation library) has been renamed with Neon due to it being a more fitting name for this library in our opinion. The name Oxygen will now be used for an upcoming web/app framework that will start development in the future.

Krypton (the encryption/decryption library) has been shelved for the time being as we work on other projects. Krypton may not return, or return as a completely different library. We may include Krypton's planned features in a future Magnesium release.

By Kassidy (@vesperyn) from Wednesware.
